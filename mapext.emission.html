class mapext.emission.emission_models.ame_lognormal(ame_ampl=10, ame_peak=27, ame_width=0.5, **kwargs)[source]

Bases: FittableEmissionModel

Emission model for an AME lognormal source.

Parameters:

  • ame_ampl (float) – AME peak flux density

  • ame_peak (float) – AME peak frequency

  • ame_width (float) – AME lognormal width

Formulaism:
\[S^{\mathrm{AME}}_\nu = A_\mathrm{AME} \cdot \exp\left\{ -\frac{1}{2}\left( \frac{\ln(\nu/\nu_\mathrm{AME})}{W_\mathrm{AME}} \right)^2 \right\}\]

For details see Stevenson (2014) .

Notes:

Either all or none of input nu, area, ame_ampl, ame_peak and ame_width must be provided consistently with compatible units or as unitless numbers.

class mapext.emission.emission_models.freeFree(ff_em=100, ff_Te=7000, **kwargs)[source]

Bases: FittableEmissionModel

Emission model for free-free UCHii emission

Parameters:

  • ff_em (float) – Free-free emission measure

  • ff_Te (float) – Free-free electron temperature

Formulism:
\[g^\mathrm{ff}_\nu = \ln\left(\exp\left\{5.90 - \frac{\sqrt{3}}{\pi}\ln\left(\left[ \frac{\nu}{\mathrm{GHz}} \right] \left[\frac{T_e}{10^4\,\mathrm{K}}\right] ^\frac{3}{2}\right)\right\} + 2.71828\right)\]
\[\tau^\mathrm{ff}_\nu = 5.468\times 10^{-2} \cdot T_e^{-\frac{3}{2}} \left[ \frac{\nu}{\mathrm{GHz}} \right]^{-2} \left[\frac{EM}{\mathrm{pc\,cm}^-6}\right] g^\mathrm{ff}_\nu\]
\[T^\mathrm{ff}_\nu = T_e \left(1-e^{-\tau^\mathrm{ff}_\nu}\right)\]
\[S^\mathrm{ff}_\nu = \frac{2k_B\Omega\nu^2}{c^2} T^\mathrm{ff}_\nu\]

For details see Draine (2011) .

Note:

Either all or none of input nu, area and ff_em must be provided consistently with compatible units or as unitless numbers.

class mapext.emission.emission_models.thermalDust(tdust_Td=20, tdust_tau=-4, tdust_beta=1.5, **kwargs)[source]

Bases: FittableEmissionModel

Emission model for the Planck modified thermal dust curve - a modified blackbody with opacity varying as frequency to some dust spectral index.

Params tdust_Td:

Thermal dust temperature

Params tdust_tau:

Thermal dust opacity (given as log_10(tau))

Params tdust_beta:

Thermal dust spectral index

Formulism:
\[S^\mathrm{td}_\nu = \frac{2k_B\Omega\nu^3}{c^2} \frac{1}{e^{h\nu/k_BT_b}-1} \cdot \tau_{\nu_0} \cdot \left(\frac{\nu}{\nu_0}\right)^\beta\]

For details see Draine and Li (2001) .

Notes:

Either all or none of input nu, area, tdust_Td, tdust_tau and tdust_beta must be provided consistently with compatible units or as unitless numbers.
